# Customer Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to identify purchasing patterns, customer segments, and revenue insights. The analysis was performed using Python, SQL, and Power BI to transform raw data into actionable business insights.

The project includes data cleaning, exploratory data analysis, SQL-based business queries, and an interactive Power BI dashboard.

---

## Dataset
The dataset contains information about customer purchases including:

- Customer demographics
- Product categories
- Purchase amount
- Discounts and promo codes
- Subscription status
- Shipping type
- Review ratings

The dataset was cleaned and analyzed to identify key trends in customer spending behavior.

---

## Tools & Technologies Used

- **Python (Pandas, NumPy)** – Data cleaning and analysis  
- **Jupyter Notebook** – Exploratory Data Analysis  
- **PostgreSQL / SQL** – Business queries and insights  
- **Power BI** – Interactive dashboard and visualizations  
- **Git & GitHub** – Version control and project hosting  

---

## Project Workflow

### 1. Data Cleaning
- Loaded dataset into Python
- Checked missing values
- Handled missing review ratings
- Standardized column names
- Created new derived features

### 2. Exploratory Data Analysis
- Customer purchase distribution
- Category-wise sales
- Discount usage trends
- Review rating patterns
- Customer demographic insights

### 3. SQL Analysis
SQL queries were used to extract business insights including:

- Revenue by gender
- Top rated products
- Shipping type comparison
- Subscribers vs non-subscribers revenue
- Discount dependent products
- Revenue by age group
- Customer segmentation analysis

All queries are stored in:
```
customer_behavior_sql_queries.sql
```

## Key Insights

- Female customers generated slightly higher total revenue.
- Express shipping users tend to spend more per purchase.
- Subscribers contribute significantly to revenue.
- Some products show heavy dependency on discounts.
- Loyal customers form the largest customer segment.

---

## Business Recommendations

- Introduce loyalty programs to retain customers.
- Promote subscription plans to increase recurring revenue.
- Optimize discount strategies to protect margins.
- Focus marketing efforts on high-performing product categories.
